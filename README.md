# word-hunter-hangman
A simple and interactive Hangman word-guessing game built using Python, where players try to guess the hidden word letter by letter with limited attempts.
 Features
 *Beautiful background screens (Loading, Menu, Category based)
 *Multiple categories:
     Food
     Animals
     Anime
 *Cartoons
 Sound effects (correct, wrong, click sounds)
 Background music support
 Particle explosion animation on win
 Smooth fade transitions
 Interactive on-screen keyboard
 Animated hangman drawing
 Back button navigation
    
 How to Play
*Click START to begin.
*Select a category.
*Guess the word using the on-screen keyboard.
*Correct guesses reveal letters.
*Wrong guesses draw parts of the hangman.
*Guess the full word before 6 wrong attempts.
